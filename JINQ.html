Javascript Intergrated Query Language

Javascript extensions, that allow array manipulation though use of simple methods

distinct

distinct removes all duplicate elements, only returning the the distinct elements.

Overloads

param1 (boolean) = byValueAndType. if false values are checked for equality of value. if true values are checked for equality of value and type

var items = [1, 2, 3, "1", "2", 3];

items.distinct();

items.distinct(false);

items.distinct(true);

contains

contains returns true or false, indicating whether the array contains a matching argument

Overloads

param1 (object) = value.
the value that is checked for equality.
param2 (boolean) = byValueAndType. if false values are checked for equality of value. if true values are checked for equality of value and type

var items = [1, 2, "3"];

items.contains("3");

items.contains(3);

items.contains(3, true);

first

gets the first element of the array

Overloads

param1 (boolean) = ignoreNullOrUndefined.
ignores all null or undefined values.

var items = [null, 1, 2, "3"];

items.first();

items.first(true);

last

gets the last element of the array

Overloads

param1 (boolean) = ignoreNullOrUndefined.
ignores all null or undefined values.

var items = [1, 2, "3", null];

items.last();

items.last(true);

allByType

gets all elements that match the given type name

Overloads

param1 (String) = typename.
the typename that should match

var items = [1, 2, "3", "a", "b", {c:1}, 0xF6];

items.allByType("string");

items.allByType("number");

items.allByType("object");
